# RecoverCall Website

A professional static marketing website for RecoverCall, a missed call recovery and Housecall Pro setup service for small HVAC companies.

## Pages

- `index.html` - Home page
- `how-it-works.html` - Process page
- `pricing.html` - Pricing and FAQ page
- `book-a-call.html` - Booking page with Calendly placeholder

## Tech

- HTML
- CSS
- Vanilla JavaScript
- No frameworks
- No build step

## How To View Locally

Open `index.html` in a browser.

## Calendly Setup

Open `book-a-call.html` and replace this placeholder:

```html

```
